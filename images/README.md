# Images Needed

Place your product images in this folder. The following images are referenced in `index.html`:

- `images/product-demo.png` - Main product demo/screenshot
- `images/before.png` - Before comparison image
- `images/after.png` - After comparison image  
- `images/trackman-logo.png` - Integration partner logo (or your logo)

You can replace these image paths in `index.html` once you have your actual images.
