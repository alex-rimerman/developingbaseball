# Developing Baseball - Professional Website

A professional website showcasing Developing Baseball's elite baseball development and training programs.

## About

This is a static website built with HTML, CSS, and JavaScript. It showcases:
- Training programs and methodologies
- About section with statistics
- Contact information and form
- Responsive, modern design

## Deployment

This is a static site that can be deployed to any static hosting service:

### Recommended: Netlify (Free & Easy)
1. Go to [netlify.com](https://netlify.com) and sign up
2. Drag and drop this folder onto the Netlify dashboard
3. Add your custom domain: `developingbaseball.com`
4. Update DNS records as instructed by Netlify

### Alternative Options:
- **Vercel**: Connect GitHub repo or drag & drop
- **Cloudflare Pages**: Free static hosting
- **GitHub Pages**: Connect your GitHub repo

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive features (mobile menu, form handling, animations)

## Local Development

Simply open `index.html` in your browser or use a local server:

```bash
python -m http.server 8000
```

Then visit: http://localhost:8000
